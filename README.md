# UCI-Credit-Card
# Credit Card Default Prediction (Logistic Regression)

This project applies Logistic Regression to predict whether a credit card
customer will default on their payment in the following month.

## Dataset
Credit card customer data including payment history and related features.

Target variable:
- Default payment next month (0 = No, 1 = Yes)

## Model
- Algorithm: Logistic Regression
- Task: Binary Classification
- Train/Test split applied

## Tools
- Python
- pandas
- scikit-learn
